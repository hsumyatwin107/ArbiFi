# AI-Powered Dispute Resolver

A hackathon project built to simulate a decentralized dispute resolution system using AI reasoning and multi-role workflows.

---

## 🚀 Features
- Multi-role system (Party A, Party B, Admin)
- AI-generated dispute decisions
- Structured workflow for case handling
- Prototype decentralized logic inspired by blockchain systems

---

## 🛠️ Tech Stack
- Next.js
- TypeScript
- Node.js
- Prisma
- PostgreSQL
- OpenAI API
- Solana concepts
